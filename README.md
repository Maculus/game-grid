## About this project

This is a gaming portal which list and provides the latest games overview, ratings and stats. It was built using React and Typescript and includes but not limited to following concepts:

- Build reusable function components
- Manage component state
- Build forms with React Hook Forms
- Build & mangage generic and custom hooks
- Implement open source UI libraries (Chakra UI V3)
- Consume an API (RAWG) on the backend
- Deploy app to production on Vercel cloud platform (https://game-grid-web.vercel.app/)

  ## Project Setup
  To get started with this project follow the steps below:
  
1. Clone this repository to your local machine.
2. Run `npm install` to install the required dependencies.
3. Run `npm run dev` to start the web server. 
3. Get a RAWG API key at https://rawg.io/apidocs. You'll have to create an account first. 
4. Add the API key to **src/services/api-client.ts**
5. Run `npm run dev` to start the web server.
